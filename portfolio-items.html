Sistema gestor de archivos

Con php, html, css, js, Mysql

Es un sistema de gestión de archivos escrito en PHP es una solución eficiente y versátil para administrar y organizar de manera efectiva los archivos en un entorno web. Utilizando el poder y la flexibilidad de PHP, este sistema permite a los usuarios realizar diversas operaciones en sus archivos, como subir, descargar, renombrar, eliminar y buscar archivos.

El sistema está diseñado con una interfaz intuitiva y fácil de usar que facilita la navegación y la gestión de los archivos. Además, se puede implementar una estructura de carpetas jerárquica para organizar los archivos de manera lógica y coherente. Esto permite a los usuarios acceder rápidamente a sus archivos y realizar acciones específicas en ellos,

Una de las principales dificultades al hacer este proyecto fue la distribución equitativa de tareas entre los miembros del equipo. Dado que cada persona tenía habilidades y conocimientos distintos, fue necesario asegurarse de asignar responsabilidades de manera justa y equilibrada.
